Drzewa decyzyjne
Drzewa decyzyjne są zaimplementowane w pakiecie rpart (nieco odmienna implementacja dostępna jest w pakiecie tree). Dla uzyskania bogatszej (i ładniejszej) wizualizacji drzew warto skorzystać z pakietu rpart.plot.

library(rpart)
library(rpart.plot)
Drzewa klasyfikacyjne
Poniższy kod wykorzystuje zbiór danych Carseats z pakietu ISLR. Będziemy klasyfikować obserwacje do dwóch klas: wysoka sprzedaż i niska sprzedaż. Uzupełniamy zbiór danych o zmienną objaśnianą w zadaniu klasyfikacji.

library(ISLR)
CarseatsH <-
  Carseats |>
  transform(High = factor(ifelse(Sales <= 8, "No", "Yes")))
head(CarseatsH)
Budujemy drzewo klasyfikacyjne do predykcji High na podstawie pozostałych zmiennych (poza Sales).

sales_high_rpart <- rpart(High ~ . - Sales, data = CarseatsH)
plot(sales_high_rpart, margin = 0.1, minbranch = 1.5)

Estymacja błędu testowego
Metodą zbioru walidacyjnego estymujemy błąd testowy dla drzewa klasyfikacyjnego w rozważanym problemie.

set.seed(1)
n <- nrow(CarseatsH)
train <- sample(n, 0.8 * n)
test <- -train

sales_high_big <- rpart(High ~ . - Sales, data = CarseatsH, subset = train)
sales_high_big_predicted <- predict(sales_high_big, newdata = CarseatsH[test,], type = "class")
table(sales_high_big_predicted, CarseatsH$High[test])
mean(sales_high_big_predicted != CarseatsH$High[test])

Do znalezienia optymalnego poddrzewa stosujemy przycinanie stosowane złożonością. Poniższy wykres pokazuje zależność błędu walidacji krzyżowej od rozmiaru poddrzewa (i malejącego parametru złożoności).

plotcp(sales_high_big)
Szczegółowe informacje pokazuje raport tekstowy. Kolumna xerror podaje wartości względnego błędu CV, a kolumna CP zawiera wartość parametru złożoności, który wykorzystywany do przycięcia drzewa.

printcp(sales_high_big)

i przycinamy duże drzewo do tego poddrzewa (czyli poddrzewa o 7 liściach).

Estymacja błędu testowego
Metodą zbioru walidacyjnego szacujemy błąd testowy (MSE).

set.seed(2025)
n <- nrow(Boston)
train <- sample(n, 0.8 * n)
test <- -train

medv_big <- rpart(medv ~ ., data = Boston, subset = train)
medv_big_pred <- predict(medv_big, newdata = Boston[test,])
mean((medv_big_pred - Boston$medv[test])^2)
Struktura dużego drzewa

rpart.plot(medv_big, digits = 4)
Wyznaczamy optymalne poddrzewo metodą przycinania sterowanego złożonością.

plotcp(medv_big)
W postaci tekstowej

medv_big$cptable
Optymalne w tym przypadku jest - jak widać - drzewo duże. Według reguły 1 SE można je jednak przyciąć do drzewa o 7 liściach.

n_min <- which.min(medv_big$cptable[, "xerror"])
limit <- sum(medv_big$cptable[n_min, c("xerror", "xstd")])
n_1se <- min(which(medv_big$cptable[, "xerror"] < limit))
cp_1se <- medv_big$cptable[n_1se - 1, "CP"]
medv_pruned <- prune(medv_big, cp = cp_1se)
rpart.plot(medv_pruned)
Estymata błędu testowego

medv_pruned_pred <- predict(medv_pruned, newdata = Boston[test,])
